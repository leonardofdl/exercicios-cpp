# Sucessor e Antecessor em C++

Programa simples desenvolvido durante curso técnico em informática para praticar lógica de programação e operações básicas com números inteiros.

## Funcionalidade
- Recebe um número inteiro do usuário
- Calcula o antecessor
- Calcula o sucessor

## Tecnologias utilizadas
- C++
- Biblioteca iostream

## Como executar
- Compile o arquivo `sucessor_antecessor.cpp` em um compilador C++
- Execute o programa via terminal ou IDE
